Escenario 2

Workspace e inbox agrupado

Pasos

  1. Entra al workspace de quotes.
  2. Revisa los grupos visibles del inbox.
  3. Usa el buscador.
  4. Prueba filtros comerciales y toggle de archivadas.
  5. Abre un caso desde el inbox.

Escenario 5

Quote review, pricing split y operator notes

Pasos

  1. Llega a un caso con shortlist y seleccion activa.
  2. Abre la vista Quote.
  3. Abre Case y escribe una operator note.
  4. Vuelve a Quote.

Checklist

Escenario 6

Compare category-specific

Pasos

  1. Desde la shortlist, selecciona hasta 5 opciones de una misma categoria.
  2. Abre el compare panel.
  3. Cambia la categoria desde el selector.
  4. Promueve una opcion nueva al active quote.

Checklist

Escenario 7

Versions y reuse as base

Pasos

  1. Haz un cambio que cree una nueva active version.
  2. Abre la vista Versions.
  3. Selecciona una version superseded.
  4. Usa Use as base si el caso lo permite.

Checklist

Escenario 8

Archive, reactivate y resume continuity

Pasos

  1. Archiva un caso desde workspace o workbench.
  2. Vuelve al inbox y confirma su estado.
  3. Reactiva el caso.
  4. Abre Conversation.

Checklist

Escenario 9

Share y export del quote activo

Pasos

  1. Con una version activa completa, abre Quote.
  2. Usa Copy shareable summary.
  3. Usa Export printable artifact.
  4. Abre la pagina del export generado.
